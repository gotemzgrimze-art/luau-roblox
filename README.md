# Luau Roblox Project

A modern Roblox game development project using Luau and Rojo.

## Project Structure

- `src/` - Main source code
- `tests/` - Test files

## Setup

1. Install [Rojo](https://rojo.space/)
2. Install [Selene](https://kampfkarren.github.io/selene/) for linting
3. Install [StyLua](https://github.com/JohnnyMorganz/StyLua) for formatting

## Development

Serve your project:
```bash
rojo serve
```

Build for release:
```bash
rojo build -o output.rbxl
```

## Tools

- **Selene**: Luau linter (configuration in `selene.toml`)
- **StyLua**: Luau code formatter (configuration in `stylua.toml`)
